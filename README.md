## 📌 Giới thiệu
Repository này lưu trữ toàn bộ bài tập thực hành và dự án nhỏ trong khuôn khổ môn học **An toàn thông tin**. 

Để thuận tiện cho việc chấm bài, các bài tập được tổ chức theo cấu trúc chuẩn:
- **src/**: Chứa mã nguồn (.cpp).
- **bin/**: Chứa file thực thi (.exe) đã biên dịch tĩnh, chạy được ngay trên Windows.
- **Click_To_Run.bat**: Script hỗ trợ cô chạy nhanh chương trình mà không cần cài đặt môi trường.
